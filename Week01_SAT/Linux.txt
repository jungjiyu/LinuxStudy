용어 정리
	: 가상머신 != 멀티부팅
		: 멀티부팅 ) 실제로 하드디스크 파티션 나누는거
		: 가상머신) 윈도우 띄어놓고 프로그램 통해 가상의 환경 생성

	:host pc) 자원 나눠주는 자리
	:guest pc) 자원 갖다 쓰는 컴터
		: 우리가 만든 가상컴터(Vmware)
	: IP 
		: 각 컴터의 고유의 주소
		: NIC (네트워크 카드)와 소통 필요
	: 프로세스) 물리적인거. cpu 같은거
	: 프로세서) 일처리할 수 있는 경로
